## 06_03_1~2
* I/O(Input/Output): 메모리-외부장치 사이에 복사 데이터가 왔다갔다 하는 것
    * 입력: 데이터가 외부 장치 -> 메모리 
    * 출력: 데이터가 메모리 -> 외부 장치 
* DMA를 사용하여 CPU 개입 없이 외부 장치(디스크)의 데이터를 메모리 주소 공간으로 복사함 
    * 실제로 입출력 데이터는 운영체제 내부로 먼저 복사된 후 -> 운영체제가 프로세스의 주소 공간으로 복샇마 
    * 운영체제 우회하여 직접 데이터를 프로세스 주소 공간에 복사할 수 있음 => zero-copy 기법 
* 입출력 블로킹 대기열, 준비 완료 대기열 2가지가 존재함 
    * 준비 완료 대기열: 대기열 안의 프로세스가 다시 실행되는 조건이 준비되었음을 의미 
    * 왜 직접 실행하지 않고 따로 있나? : 실행할 준비가 완료된 프로세스가 여러개 있고, 할당되지 못한 스레드들이 여러개 있을 수 있으므로 대기열이 필요
