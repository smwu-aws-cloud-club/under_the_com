# chap 06_1 정리 
26.02.11 update

## 06_1_1~7
* 장치 레지스터: 장치에 관련된 일부 정보를 저장함 
    1. 데이터 저장 레지스터
        * e.g., 사용자가 키보드의 키 누를때 저장됨
    2. 제어 정보와 상태 정보를 저장하는 레지스터 
        * 레지스터에 읽고쓰기 작업 통해 장치 제어하거나 장치 상태 볼 수 있음 
* 입출력 명령어(input and output instruction): 장치 레지스터를 읽고 쓰는 특정 기계 명령어
    * e.g., x86의 IN, OUT 기계 명령어 
* `메모리 사상 입출력(memory mapping input and output)`
    * (메모리) 주소 공간의 일부분을 장치에 할당하여 메모리를 읽고/쓰는 것처럼 장치를 제어하는 방법 
<br>

* 장치 상태 레지스터: 장치의 작업 상태를 기록. 이 레지스터 값을 읽음으로써 cpu는 장치를 읽을 수 있는지 쓸수 있는지 판단 가능 
* 폴링(polling) 
    * 동기식 설계 방식: cpu는 누군가 키를 누를 때까지 계속 검사하면서 대기함 => 비동기로 바꿔서 최적화하자 
* 인터럽트 구동식 입출력(Interrupt driven input and output): 입출력을 비동기로 처리하는 방법
