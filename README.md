## 📖 Study Rules 
* 시험기간을 제외한 매주 정해진 분량을 읽고, 각자 원하는 방식으로 내용을 정리한다 (+궁금한점, 추가로 찾아본 점도 포함)
* 해당 레포지토리에 각자 자기 이름의 branch를 생성한 다음, 자기 이름의 디렉토리를 생성한다
* 내용정리는 각자 자기 이름의 디렉토리 안에서 진행해야한다 
* 매주 스터디 테크톡 시간 전까지 PR 하여 일괄적으로 main 브랜치로 merge하도록 한다
  * PR 제목은 `[자기이름] n주차 스터디` 로 한다
* 벌금 제도 
  * 미흡한 경우: 1000원
  * 미제출한 경우: 2000원

## 📖 Study Rules 
- 스터디원 모두 정해진 분량을 읽고 개인 블로그, 노션 등에 정리 후 정리한 내용을 포함한 PR을 올려 인증 **(인증 마감: 스터디 당일 18:00)**
- PR에는 다음 내용이 담겨 있어야 합니다.
    - 새롭게 알게된 점(New)
    - 어려웠거나 이해하지 못한 부분(Difficulty)
    - 추가 내용(Amendment)
- 스터디 시간에 **랜덤으로 2명**을 선정하여 정리한 내용에 대한 발표 진행
    - 연속 2주 선정 시 재추첨 (1주차 발표 후 2주차에는 발표 X, 3주차에는 발표 O)
    - 해당 주차의 앞부분/뒷부분 내용으로 나눠 발표 진행
    - 발표 자료 만들어오실 필요 없습니다! 정리된 내용 화면 공유하면서 발표해주시면 돼요
- 발표 진행 후 알게 된 내용들에 간단한 테크톡의 시간(10분 - 20분 정도)
- 스터디 시간은 총 60분이며, 앞부분 발표(20분), 뒷부분 발표(20분), 테크톡(10-20분)으로 진행됩니다. **정해진 시간을 초과할 시 양해를 구하고 다음 순서를 진행할 수 있습니다.**
